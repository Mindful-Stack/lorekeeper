# General Standards

This is the entry point for cross-repo standards. Add files here for things that apply
to every project: code review checklists, commit message conventions, testing philosophy,
documentation expectations.

Example file ideas:
- `pr-review-checklist.md`
- `commit-conventions.md`
- `testing-philosophy.md`
